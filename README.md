# erc-pow
erc-pow挖矿工具

编译好的软件：https://github.com/longtaolii/erc-pow/releases/tag/v0.1

使用方法：

1、将config.json与程序放在同目录下

2、config.json替换你的公、私钥，并且在tokens配置你想要挖的币

3、ercpow.exe -h查看使用方法，-address和-token是必须配置的并且必须是在配置文件里有的.-test可以测试，不会花钱发送交易
